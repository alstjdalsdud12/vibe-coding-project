# 앱 실행 흐름

이 문서는 앱을 처음 보는 사람도 전체 흐름을 이해할 수 있도록 작성되었습니다.

---

## 한 줄 요약

> 유저가 캐릭터 정보를 입력하면, AI가 고유한 캐릭터를 자동으로 만들어주고, 그 캐릭터로 바로 게임을 플레이할 수 있다.

---

## 앱의 구성 요소

이 앱은 3개의 파트로 나뉩니다.

```
① Unity (모바일 앱)     — 유저가 실제로 보고 조작하는 화면
② Node.js 백엔드 서버   — Unity와 AI/Firebase 사이를 연결하는 중간 서버
③ 외부 서비스           — Claude AI (캐릭터 생성), Firebase (데이터 저장)
```

Unity는 직접 Claude나 Firebase에 접근하지 않고, 반드시 백엔드 서버를 통해서만 통신합니다.
이유는 API 키 같은 민감한 정보를 앱 안에 넣으면 보안상 위험하기 때문입니다.

---

## 흐름 1 — 캐릭터 만들기

```
1. 유저가 앱을 열고 "새 캐릭터 만들기"를 누릅니다.

2. 캐릭터 생성 화면에서 3가지를 입력합니다.
   - 외형/생김새  예) "검은 머리, 키 크고 우아함"
   - 무기        예) "지팡이"
   - 컨셉        예) "신중하고 차분한 마법사"

3. "생성" 버튼을 누르면 로딩 화면이 뜨고,
   Unity가 백엔드 서버로 입력 내용을 전송합니다.

4. 백엔드 서버가 Claude AI에게 입력 내용을 보냅니다.
   Claude는 이것을 분석해서 아래 내용을 자동으로 만들어 줍니다.
   - 캐릭터 이름    예) "실바나 아쉬크로프트"
   - 스탯          예) HP 85 / 공격력 35 / 방어력 25 / 마나 120
   - 특수능력       예) "정적의 화살 — MP 20 소모, 적에게 ATK×1.8 피해"
   - 배경 스토리    예) "오래된 숲의 끝자락에서 홀로 수련한 마법사..."

5. 백엔드 서버가 만들어진 캐릭터를 Firebase(데이터베이스)에 저장합니다.
   저장이 완료되면 캐릭터 데이터를 Unity로 돌려줍니다.

6. Unity 화면에 완성된 캐릭터 정보가 표시됩니다.
   유저는 이 화면에서 "게임 시작"을 눌러 바로 플레이할 수 있습니다.
```

---

## 흐름 2 — 이전에 만든 캐릭터 불러오기

```
1. 유저가 앱을 열면 메인 메뉴에 이전에 만든 캐릭터 목록이 뜹니다.
   (Firebase에 저장된 데이터를 백엔드 서버를 통해 불러옵니다.)

2. 목록에서 캐릭터를 탭하면 상세 정보를 볼 수 있습니다.
   - 스탯 수치, 특수능력 설명, 배경 스토리 확인 가능

3. "게임 시작"을 누르면 해당 캐릭터로 전투 씬이 시작됩니다.

4. 필요 없는 캐릭터는 삭제할 수 있습니다.
   (Firebase에서 데이터가 완전히 삭제됩니다.)
```

---

## 흐름 3 — 전투 플레이

```
1. 전투 화면이 시작되면 내 캐릭터와 적이 등장합니다.
   화면에는 각자의 이름과 HP바가 텍스트로 표시됩니다.
   (이미지 없이 텍스트만으로 표현합니다.)

2. 내 턴이 되면 3가지 행동 중 하나를 선택합니다.
   ┌─────────────────────────────────────────┐
   │  [일반 공격]  [스킬 사용]  [도망치기]   │
   └─────────────────────────────────────────┘

3. 각 행동의 결과가 전투 로그에 문장으로 출력됩니다.
   예) "실바나가 정적의 화살을 쏜다! 적에게 63 피해!"
   예) "적이 반격한다! 실바나에게 18 피해!"

4. 데미지 계산 방식
   - 일반 공격: 공격자 공격력 - 방어자 방어력 = 피해량 (최소 1)
   - 스킬 사용: 마나를 소모하고, 스킬별 효과 적용

5. 전투 종료 조건
   - 적 HP가 0 이하 → 스테이지 클리어 화면
   - 내 HP가 0 이하 → 게임 오버 화면
   - 도망치기 선택  → 메인 메뉴로 복귀

6. 클리어 또는 게임 오버 후 메인 메뉴로 돌아갑니다.
```

## 전체 흐름 한눈에 보기

```
앱 실행
  │
  ▼
메인 메뉴 ──────────────────────────────┐
  │                                     │
  │ 새 캐릭터 만들기                      │ 기존 캐릭터 선택
  ▼                                     ▼
캐릭터 생성 화면               캐릭터 상세 화면
  │ 외형/무기/컨셉 입력           │ 스탯/능력/스토리 확인
  │                              │
  ▼                              │
백엔드 서버                      │
  ├─▶ Claude AI → 캐릭터 생성    │
  └─▶ Firebase → 데이터 저장     │
  │                              │
  ▼                              │
캐릭터 결과 화면 ◀───────────────┘
  │ 이름/스탯/능력/스토리 표시
  │ "게임 시작" 버튼
  ▼
전투 씬
  │ 일반 공격 / 스킬 / 도망
  │
  ├─▶ 승리 → 클리어 화면 → 메인 메뉴
  └─▶ 패배 → 게임 오버 화면 → 메인 메뉴
```
